Jeu vidéo 2D — MonoGame

Description

Plateformer 2D en C# (.NET 8) avec MonoGame DesktopGL : mouvements (saut, dash), tirs, IA simple des ennemis, particules et gestion audio (musique d’ambiance, effets).

Technologies

  • C# / .NET 8 — MonoGame DesktopGL
  • Sprites 2D, particules, caméras et collisions
  • Audio : SoundEffect / Song (jump, dash, shoot, explosion, background)

Essayer le projet

Télécharger (Windows) Remarque : le jeu est une application native (MonoGame). Le navigateur télécharge l'archive — il ne peut pas l'exécuter directement. Si le fichier n'existe pas, suivez les instructions ci‑dessous pour le construire.

Instructions rapides : construire et lancer (Windows)
  1. Ouvrir la solution projets\Jeu2D\Jeu2D.sln dans Visual Studio.
  2. Construire en Release pour la configuration Windows (Build → Build Solution).
  3. Récupérer l'exécutable dans projets\Jeu2D\bin\Debug\net8.0 (ou \bin\Release\net8.0).
  4. Créer une archive ZIP contenant les fichiers nécessaires et placer la ZIP dans projets\Jeu2D\Jeu2D.zip pour activer le lien de téléchargement.

Alternativement, lancez le jeu depuis Visual Studio (F5) ou utilisez dotnet run --project projets\Jeu2D\Jeu2D.csproj depuis PowerShell pour tester sans packaging.

Contrôles

  • A / D : gauche / droite — Espace : saut — Shift : dash — K : tirer
  • Échap : quitter
