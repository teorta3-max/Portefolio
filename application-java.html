Application Java — Gestion d'étudiants

Description

Application Java orientée objet pour gérer une liste d’étudiants via une interface graphique. Architecture DAO pour séparer la logique métier et l’accès aux données.

Fonctionnalités

  • UI (Swing/JavaFX selon implémentation) dans FenetrePrincipale.java
  • Modèle : Etudiant.java ; Accès aux données : EtudiantDAO.java
  • Point d’entrée : Main.java

Essayer le projet

Télécharger (JAR) Télécharger (Source ZIP) Si le JAR n'est pas présent, téléchargez l'archive source ci‑dessous ou suivez les instructions pour construire le JAR localement.

Instructions rapides : construire le JAR (Windows)
  1. Installer le JDK (Java 11+).
  2. Ouvrir PowerShell dans projets\ApplicationJava.
  3. Compiler les sources : javac -d bin Main.java dao\*.java model\*.java ui\*.java (ou utilisez votre IDE).
  4. Créer le JAR : jar cvfe ApplicationJava.jar Main -C bin ..
  5. Copiez ApplicationJava.jar dans projets\ApplicationJava pour que le lien de téléchargement fonctionne.

Pour exécuter le JAR localement : java -jar ApplicationJava.jar ou double‑cliquer si Java est associé.
